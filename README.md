이름 : 김문수

소속 : 소프트웨어 마에스트로 7기

1단계 모바일 앱 만들기 과제 repository입니다.

Application Name : Secretter (Secret + Letter)

Secretter는 특정 위치에서 편지를 남기게 되면,
편지가 있는 위치에서 편지를 읽을 수 있는
위치 기반의 편지 Application 입니다.

Client : Android Studio

Server : Spring Framework, JSP, Mybatis, MySQL, AWS, Apache Tomcat8.0

기능.

1. 앱과 서버 통시하는 로직 포함.

2. 게임 첫 설치 시간, 접속 시간, 앱 종료시간 DB에 저장.

3. 페이스북 로그인, 친구 목록 리스트 (친구 자동 추가), Daum 지도, 카카오톡 공유 연동.

4. 앱 스토어 업로드 
  https://play.google.com/store/apps/details?id=onepercent.mobile.com.onepercent

5. AWS (Amazon Web Server EC2) 배포

6. 데이터는 MySQL에 저장되어 검색, 삽입, 삭제, 갱신 가능

7. Spring Framework 및 JSP 사용


추가 기능.

1. 친구 자동 추가, 초대하기 기능.

2. 푸시 알림 기능.

3. 관리자가 데이터를 편리하게 관리할 수 있는 CMS(관리시스템) 구축
